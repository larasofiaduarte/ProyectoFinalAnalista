# Sistema para Salón de Belleza
Este proyecto es un sistema de gestión para una pequeña empresa, Salón de Belleza H.Y. 
Desarrollado en Java utilizando Maven para la gestión de dependencias y Swing para la interfaz gráfica, con base de datos MySQL gestionada a través de WAMPServer y el framework de persistencia JPA para la interacción con la base de datos.

# Características del Sistema
El sistema permite gestionar las siguientes áreas clave de la empresa:

- Gestión de Caja: Control de ingresos y egresos, resumen de operaciones diarias.
- Gestión de Proveedores: Alta, baja, modificación y consulta de proveedores.
- Gestión de Empleados: Administración de empleados, roles, y sus horarios.
- Gestión de Clientes: Manejo de clientes, historial de servicios y contacto.
- Gestión de Productos: Inventario de productos, control de stock y precios.
- Gestión de Turnos: Asignación y control de turnos para empleados y clientes.
